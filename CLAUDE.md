# 社交面具之下：你的暗黑人格成分鉴定

## 项目简介
面向 18-28 岁年轻群体的娱乐向心理学测试 H5。通过 20 道高压场景化问答，探索用户在社会道德表象下压抑的负面特质，生成暗黑赛博风格的结果报告。

## 技术栈
- 纯 HTML5 + CSS3 + JavaScript（Vanilla JS）
- 雷达图：Canvas 自绘（无第三方依赖）
- 分享截图：html2canvas（CDN 加载）
- 字体：Google Fonts（Cinzel Decorative 英文标题 + 系统黑体中文）

## 文件结构
```
index.html        — 主入口，三视图（首页/答题/结果）
css/style.css     — 暗黑赛博风格样式 + 动画
js/data.js        — 题库数据（20题）、维度配置、结果文案
js/app.js         — 核心逻辑（答题流程、计分、结果渲染、雷达图、交互）
```

## 构建规范
- 所有结果数据全免费即时解锁，无付费点
- Canvas 绘制雷达图，不引入第三方图表库
- 配色严格遵循 spec 定义的色彩矩阵
- 响应式设计，适配手机端（375px-428px 主流宽度）
- 选项点击后禁用防重复提交，答完自动计分

## 部署方式
纯静态页面，双击 index.html 或部署到任意静态服务器/CDN。
后续可迁移至微信小程序（需将逻辑移植到小程序框架）。
